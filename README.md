# Full-Stack App with FastAPI and React

## Overview

This is a simple full-stack application built using FastAPI for the backend and React for the frontend. The application allows users to submit items with a name and price, and it displays the submitted items.

## Tech Stack

- **Backend**: FastAPI
- **Frontend**: React
- **Database**: PostgreSQL
- **Containerization**: Docker
- **HTTP Client**: Axios (for frontend)

## Prerequisites

- Python 3.7 or higher
- Node.js (v16 or higher)
- npm (Node Package Manager)
- Docker and Docker Compose
